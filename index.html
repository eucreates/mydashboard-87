One-Time Setup

Equilibrium uses Firebase for secure accounts and cross-device sync.
Paste your Firebase config below to get started.

How to get your Firebase config (free, 5 minutes):

1
Go to console.firebase.google.com → Create a new project
2
Click Build → Realtime Database → Create database → Start in test mode
3
Click Build → Authentication → Get started → Enable Email/Password sign-in
4
Go to Project Settings (gear icon) → scroll to "Your apps" → Add a web app → copy the firebaseConfig object
5
Paste the entire config JSON below and click Save

This config is stored only in this browser. You'll need to re-enter it on each new device where you deploy the app.
